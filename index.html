Project Overview

This website documents my eco-column project, a contained ecosystem designed to model how living organisms interact with each other and with Earth systems. The eco-column includes both aquatic and terrestrial components and demonstrates ecological principles from Units 1–4.

Ecosystem Type: Freshwater pond and forest-floor hybrid ecosystem
